GalleriomJS

Modern photo gallery

written entirely in native JavaScript

GalleriomJS speed of light

  • Production files
  • Development files

What?

The GalleriomJS gallery tries to optimize the way that you interact with DOM and leverage the GPU as much as posible.

Why?

Very easy to setup. In just about 2 minutes you're done!

For as quick as posible just pass the ID, width and height.

But don't hesitate in using others professional features.
Gallery support responsive view, thumbnail view, caption, preview

Responsive

Mobile first? No problem. Touch, swipe, samller screen..

It can handle any type of device.

Just drop it into your web page or project,
setup and refres the browser.

How simple can it be?

Simple

As small as posible.

Especially minified version(production) is very lightweight.

Install

Install CSS file in head tag

<link rel="stylesheet" href="path-to-galleriom-css">

Install JS file before closing body tag

<script src="path-to-galleriom-js">

Basic setup

// Give div element id="galleriom-jsgallery-app", set width and height

<div id="galleriom-jsgallery-app" data-width="300" data-height="300">

// And insert your photos with class="galleriom-img"

<div id="galleriom-jsgallery-app" data-width="300" data-height="300">

<img class="galleriom-img" src="assets/img/1.jpg">

<img class="galleriom-img" src="assets/img/2.jpg">

<img class="galleriom-img" src="assets/img/2.jpg">

</div>

Responsive setup

// Set width and/or height with metric units

<div id="galleriom-jsgallery-app" data-width="100%" data-height="300">

<img class="galleriom-img" src="assets/img/1.jpg">

<img class="galleriom-img" src="assets/img/2.jpg">

<img class="galleriom-img" src="assets/img/2.jpg">

</div>

Thumbnails ON

// Activate thumbs preview

<div id="galleriom-jsgallery-app" data-width="400" data-height="300" data-thumbs="true">

<img class="galleriom-img" src="assets/img/1.jpg">

<img class="galleriom-img" src="assets/img/2.jpg">

<img class="galleriom-img" src="assets/img/2.jpg">
